快速开始

三步上手 LegalWork

命令行、桌面端两种方式，灵活适配你的工作流。

Terminal 
# 安装依赖
pip install -r requirements.txt

# OCR 依赖（可选）
pip install paddleocr pytesseract pymupdf

# 运行 OCR
python3 ocr_agent.py auto 文档.pdf

# 运行脱敏
python3 redact_agent.py 文档.docx

# 运行技能
python3 run_skill.py legal-case-analysis 案件材料.pdf
Terminal 
cd apps/desktop-legalwork

# 安装依赖
npm install

# 开发模式
npm run dev

# 构建
npm run build

# 启动服务
legalwork serve --data-dir ~/.legalwork

常见问题

FAQ

关于 LegalWork 的常见疑问，快速解答你的困惑。

LegalWork 是什么？适合谁使用？

LegalWork 是面向法律专业人士的开源 AI 工作台，集成 1000+ 法律 AI 技能、智能 OCR、文件脱敏、案件管理、微信与飞书聊天接入等功能。适合律师、法务、法学院学生、法律研究者等需要处理法律文档、进行法律分析的用户。

LegalWork 是免费的吗？

是的，LegalWork 核心代码开源且免费使用。Python 技能引擎部分采用 MIT License，桌面端应用基于 PolyForm Noncommercial License。个人学习、研究、实验、教学均可免费使用。

我的案件数据安全吗？

非常安全。LegalWork 支持本地优先运行，所有文档处理、AI 推理都可以在本地完成。案件数据、客户材料不会上传到任何第三方服务器。你拥有对数据的完全控制权。

支持哪些 AI 模型？

LegalWork 采用模型无关设计，支持 DeepSeek API、OpenAI 兼容 API（如 GPT-4、Claude 等），以及任何符合 OpenAI API 格式的模型服务。你可以根据任务需求自由切换模型。

如何安装和使用？

命令行方式：安装 Python 依赖后，使用 python3 run_skill.py <技能名> <文件> 调用技能。桌面端方式：下载对应系统的安装包，启动后即可获得完整的可视化 AI Agent 工作体验。详见 GitHub 文档。

可以贡献代码或提出建议吗？

当然可以！LegalWork 是开源项目，欢迎通过 GitHub Issues 提交反馈、通过 Pull Request 贡献代码。无论是技能开发、Bug 修复、文档改进还是新功能建议，都欢迎参与。
