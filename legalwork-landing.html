快速开始

三步上手 LegalWork

支持命令行、桌面端两种使用方式，灵活适配你的工作流。

安装环境

克隆仓库并安装 Python 依赖，或使用桌面端安装包。

pip install -r requirements.txt
pip install paddleocr pytesseract

运行技能

通过命令行调用 70 项法律 AI 技能，处理文档或进行分析。

python3 run_skill.py legal-case-analysis 案件.pdf

桌面端体验

启动桌面端应用，获得完整的可视化 AI Agent 工作体验。

cd apps/desktop-legalwork
npm install && npm run dev
