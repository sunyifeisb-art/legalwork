# 合同风险审查

## 概述

| 项目 | 内容 |
|------|------|
| **能力名称** | 合同风险审查 |
| **核心功能** | 对合同文本进行系统性的法律风险识别、风险评估和修改建议输出 |
| **适用领域** | 商业合同审查、格式条款审查、重大交易合同审查、跨境合同审查 |
| **审查维度** | 合同主体、标的条款、价格与支付、履行条款、违约责任、争议解决、保密条款、知识产权、不可抗力、解除终止 |
| **风险等级** | critical（致命风险）、high（重大风险）、medium（中等风险）、low（轻微风险） |
| **关键原则** | 全面覆盖（审查所有条款）、风险导向（聚焦高风险条款）、可操作（修改建议具体可行）、平衡性（兼顾商业与法律） |

## 法律声明

> **重要提示：** 本技能提供的合同风险审查为辅助性法律分析，不构成正式法律意见。使用者应注意：第一，合同审查的深度应当根据交易的性质、金额和风险状况确定。第二，审查结论中的风险评级和修改建议仅为初步判断，最终的风险评估应当由执业律师结合具体交易背景完成。第三，合同中的空白条款、引用文件、附件等同样具有法律约束力，应当一并审查。第四，跨境合同还应当考虑法律适用、争议解决和管辖法院等特殊问题。第五，合同审查具有时效性，相关法律法规的变化可能影响审查结论的有效性。

## 一、核心概念体系

### 1.1 合同风险的类型

合同风险可以从不同角度进行分类。按风险来源划分，可以分为法律风险（合同条款违反法律强制性规定导致无效或可撤销）、商业风险（合同条款对己方不利导致商业利益受损）、履行风险（对方履约能力不足或违约风险）和程序风险（争议解决条款约定不明导致的诉讼风险）。

按风险严重程度划分，可以分为致命风险（critical），指会导致合同无效、重大损失或无法实现合同目的的风险，如违反法律强制性规定、约定违法违约责任等，必须修改才能签署。重大风险（high），指对己方利益有显著不利影响的风险，如违约责任严重不对等、争议解决条款明显不利等，应当争取修改。中等风险（medium），指存在一定风险但尚可接受的风险，如部分条款表述不够精确、存在一定歧义等，建议修改但非强制。轻微风险（low），指对合同影响轻微的风险，如格式不统一、文字表述不严谨等，可视情况决定是否修改。

### 1.2 合同审查的基本框架

合同审查应当遵循系统化的框架，按以下层次递进进行：

第一层是合同效力审查，确认合同主体是否具备相应的民事权利能力和民事行为能力，合同内容是否违反法律强制性规定和公序良俗，合同形式是否符合法定要求。

第二层是合同条款完整性审查，检查合同是否包含了交易所需的全部核心条款，包括标的、数量、质量、价款、履行期限、履行地点和方式、违约责任、争议解决等。

第三层是合同条款公平性审查，分析各条款对双方权利义务的配置是否公平合理，是否存在显失公平的格式条款，是否存在权利义务严重不对等的情况。

第四层是合同条款明确性审查，检查条款表述是否清晰明确，是否存在歧义或漏洞，定义是否一致，引用的标准和规范是否明确。

### 1.3 各类型合同的审查重点

不同类型的合同有不同的审查重点。买卖合同重点审查标的物的规格型号、质量标准、交付方式、验收程序、风险转移时点和所有权保留条款。服务合同重点审查服务范围和标准、服务期限、服务人员的资质要求、SLA（服务水平协议）和验收标准。技术开发合同重点审查技术成果的归属、知识产权的分配、后续改进的处理、技术保密和竞业限制。租赁合同重点审查租赁物的范围和使用限制、租金调整机制、维修责任的分配、提前解约的条件和违约责任。劳动合同重点审查工资标准、工作内容、工作地点、社保缴纳、竞业限制和保密义务。

## 二、操作步骤

### 2.1 第一步：合同背景了解

审查合同前应当充分了解合同的商业背景和交易目的。包括合同签订的商业目的、各方的谈判地位和议价能力、行业惯例和特殊要求、相关法律法规的适用情况。背景了解有助于确定审查的重点和修改的策略。

### 2.2 第二步：合同结构概览

通读合同全文，了解合同的整体结构和主要条款安排。标注关键词和重点关注条款，识别合同中的定义条款、核心交易条款、风险分配条款和程序性条款。对合同的结构完整性进行初步评估，判断是否存在遗漏重要条款的情况。

### 2.3 第三步：逐条审查与分析

对合同的每一条款进行逐条审查，识别其中存在的法律风险和商业风险。对于每个风险点，记录风险描述、风险等级、法律依据和修改建议。审查过程中应当注意条款之间的协调性和一致性，避免出现前后矛盾的情况。

### 2.4 第四步：综合评估与建议

对所有审查发现进行汇总分析，形成综合评估结论。确定需要优先修改的条款和可以接受的条款。提出具体的修改建议和谈判策略。出具合同审查意见书。

## 三、十大审查维度详解

### 3.1 合同主体审查

审查对方当事人的主体资格，包括是否依法设立并有效存续、是否具备从事相关业务的资质和许可、是否有权签署合同、代理人的代理权限是否充分。关注对方企业的注册资本、经营状况、信用记录和涉诉情况。

### 3.2 标的条款审查

审查合同标的条款是否明确具体，包括标的物的名称、规格型号、质量标准、数量、产地等。对于服务类合同，应当明确服务范围、服务内容、服务标准和可交付成果。标的不明确是合同争议的常见原因。

### 3.3 价格与支付条款审查

审查价格条款是否明确，包括计价方式、币种、税率、调整机制等。审查支付条款，包括支付方式、支付时间、支付条件、发票开具等。注意预付款的比例是否合理、尾款的支付条件是否明确。

### 3.4 违约责任条款审查

审查违约责任的设定是否公平合理，包括违约情形、违约金比例、损失赔偿范围、责任限制和免责条款。注意违约金是否过高（超过实际损失的30%可能被法院调减）。审查赔偿范围是否包括间接损失、预期利益损失等。

### 3.5 争议解决条款审查

审查争议解决条款是否明确有效，包括选择诉讼还是仲裁、管辖法院或仲裁机构的具体名称、法律的适用、送达地址等。诉讼和仲裁各有利弊，应当根据交易的具体情况选择。管辖条款的明确性是实务中常见的问题。

### 3.6 保密条款审查

审查保密条款的保密范围、保密期限、保密义务的例外情形、违反保密义务的违约责任。注意保密义务是否相互、保密信息的定义是否过于宽泛或过于狭窄。

### 3.7 知识产权条款审查

审查知识产权归属条款，包括合同履行过程中产生的知识产权的归属、背景知识产权的许可使用、知识产权侵权的责任承担等。技术开发合同、委托创作合同等应当特别关注知识产权条款。

### 3.8 不可抗力条款审查

审查不可抗力的定义范围是否合理、不可抗力发生后的通知义务、免责的范围和期限、合同是否自动解除或延期履行。注意常见的不可抗力事件是否被涵盖。

### 3.9 解除和终止条款审查

审查合同的解除条件、提前终止的条件和程序、终止后的义务（如返还资料、结算费用、保密义务的延续等）。注意解除权是否对等、终止通知的期限是否合理。

### 3.10 格式条款和附件的审查

对于格式合同，应当重点审查是否存在合同法规定的无效格式条款情形。合同的附件、补充协议和引用文件应当与主合同一并审查，确保与主合同的条款协调一致。

## 四、合同审查意见书模板

合同审查意见书应当包括以下部分：审查基本信息（合同名称、版本日期、审查人、审查日期）、总体评价（合同的主要风险评估摘要）、逐条审查意见（按条款序号逐一列明风险点和修改建议）、修改后风险评估（修改后的剩余风险）和特别提示（其他需要关注的事项）。

## 五、质量检查清单

效力检查：合同主体适格、意思表示真实、内容不违反强制性规定、形式合法。
内容检查：核心交易条款完整、定义清晰、权利义务明确、风险分配合理。
程序检查：签署程序合法、授权文件完备、审批手续齐全。
附件检查：附件与主合同一致、引用文件完整、所有附件已签署。

## 六、关联技能

| 关联技能 | 关系 | 说明 |
|----------|------|------|
| 合规审查 | 并行 | 需要结合合规要求审查合同条款 |
| 尽职调查 | 上游 | 交易前需了解对方资信状况 |
| 法律研究 | 上游 | 审查需要确认相关法律规范的有效性 |
| 法律文书起草 | 下游 | 审查意见中的修改建议涉及文书起草 |
| 谈判支持 | 下游 | 审查意见为合同谈判提供依据 |

## 七、局限性

第一，合同审查的效果依赖于法律规范的准确理解和适用，法律规范的变化可能影响审查结论的有效性。第二，审查结论中的风险判断具有一定的专业性，不同律师可能对同一条款有不同的风险判断。第三，商业风险和纯法律风险有时难以截然分开，需要结合具体的商业背景进行综合判断。第四，部分风险可能在合同文本层面无法完全消除，需要结合实际的履行管理来控制。第五，跨境合同的审查还面临法律适用的不确定性、语言翻译的准确性等问题。
