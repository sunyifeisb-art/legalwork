# 法律评估报告生成

## 概述

| 项目 | 内容 |
|------|------|
| **能力名称** | 法律评估报告生成 |
| **核心功能** | 对案件或法律问题进行系统性评估，生成结构化的法律评估报告 |
| **适用领域** | 诉讼案件评估、交易风险预评估、合规风险评估、项目可行性评估 |
| **关键原则** | 客观性、全面性、审慎性、可操作性 |

## 法律声明

> **重要提示：** 本技能生成的法律评估报告为辅助性分析工具，不构成正式法律意见。使用者应注意：第一，评估结论基于评估时点的法律法规和已知事实，后续发生变化可能影响评估结论的有效性。第二，不同评估者对同一法律问题可能有不同的专业判断。第三，评估报告中的风险评级为相对判断，不构成对实际结果的预测。第四，涉及重大决策的评估，建议由执业律师进行进一步审核。第五，报告中的事实陈述以当事方提供的信息为准，未经验证的事实应当标注说明。

## 一、核心概念体系

### 1.1 法律评估的类型

按评估目的和场景，法律评估可以分为以下几类：诉讼案件评估，对诉讼案件的事实基础、法律依据、胜诉可能性和风险因素进行系统评估，为当事人决策提供参考。交易风险评估，对拟进行的商业交易（股权收购、资产并购、合资合作等）中的法律风险进行识别和评估。合规风险评估，对企业经营行为的合规状况进行系统评估，识别合规风险点并提出整改建议。项目可行性评估，对特定项目的法律可行性进行评估，分析项目存在的法律障碍和解决路径。

按评估深度，可以分为初步评估和深度评估。初步评估在有限信息和时间内对法律问题进行快速判断，适用于初步决策和风险筛选。深度评估在充分收集信息和文件的基础上进行全面的法律分析，适用于重大决策前的最终确认。

### 1.2 法律评估的基本框架

法律评估应当遵循以下系统化框架：第一是事实梳理阶段，全面收集和整理与评估事项相关的事实信息，建立事实基础。第二是法律检索阶段，检索与评估事项相关的法律法规、司法解释、裁判案例和行政规范。第三是分析论证阶段，将事实与法律相结合，运用法律解释和推理方法进行分析论证。第四是风险评估阶段，识别法律风险点，评估风险发生的可能性和影响程度。第五是结论建议阶段，形成评估结论，提出具体可行的建议方案。

### 1.3 风险评估的方法论

风险评估可以采用定性分析和定量分析相结合的方法。定性分析主要依靠专业判断和经验，对风险的性质、发生条件和影响程度进行描述性评估。定量分析通过数据分析模型，对风险发生概率和损失金额进行量化评估。

风险等级通常按照三级或五级制划分。三级制包括高、中、低三个等级。五级制包括极高、高、中、低、极低五个等级。风险评估应当结合具体事项的性质、金额和影响来确定风险等级。

## 二、操作步骤

### 2.1 第一步：确定评估范围和目标

首先明确评估的具体对象和范围，包括评估涉及的法律领域、评估的时间范围、评估的地理范围（涉及的管辖区域）等。与客户或委托方确认评估的核心关切和期望目标，明确评估报告的主要受众和用途。制定评估工作计划和时间表。

### 2.2 第二步：事实收集与梳理

通过尽职调查清单、文件审阅、访谈等方式，全面收集与评估事项相关的事实信息。对收集的事实进行分类整理，建立事实清单。标注事实的确认状态（已确认、待核实、有争议、缺失）。对缺失的关键事实，列出待补充清单。

### 2.3 第三步：法律检索与研究

围绕评估事项涉及的法律问题，进行系统性的法律检索。检索相关法律法规、司法解释、部门规章和行业规范。检索相关裁判案例和行政执法案例。对检索结果进行分析整理，形成法律依据清单。

### 2.4 第四步：分析与评估

将事实与法律相结合，对每个法律问题进行逐项分析。分析应当遵循从法律到事实的逻辑路径：首先确定适用的法律规则和标准，然后将案件事实与法律要件进行对应，最后得出法律判断。对于存在不确定性的问题，应当充分分析各种可能的结果和条件。

### 2.5 第五步：撰写评估报告

按照标准结构撰写评估报告。报告应当结构清晰、逻辑严密、结论明确。风险评估部分应当包括风险点描述、风险评级、发生可能性和影响评估。建议部分应当具有可操作性，提供多种可选方案供委托方决策。

## 三、输出规范

### 3.1 标准报告结构

概要部分包括评估对象、评估目的、评估时间、主要结论的简要总结。事实概要部分阐述评估事项的事实背景、关键事实要素、事实确认状态。法律分析部分逐项分析评估涉及的法律问题，引用法律规范和分析论证。风险评估部分识别主要风险点，进行风险评级和分析。结论与建议部分给出总体评估结论、具体建议方案和风险应对措施。附注部分说明评估的局限性、使用限制和免责声明。

### 3.2 风险矩阵

风险矩阵以表格形式呈现，列明每个风险点的编号、名称、风险描述、发生可能性、影响程度、综合评级和应对建议。综合评级采用三级制：高风险（需要立即采取措施）、中风险（需要关注并在合理期限内处理）、低风险（在常规管理范围内处理）。

## 四、质量检查清单

事实基础检查：评估事项的事实背景已充分了解，关键事实要素已收集，事实确认状态已标注，缺失信息已列出待补充清单。法律分析检查：适用的法律规范已检索完整，法律分析逻辑严密，不确定性因素已充分讨论。风险评估检查：风险点识别全面，风险评级合理，风险应对建议具有可操作性。报告完整性检查：报告结构完整，结论明确，免责声明充分。

## 五、关联技能

| 关联技能 | 关系 | 说明 |
|----------|------|------|
| 法律研究 | 上游 | 评估需要法律检索和研究 |
| 事实抽取 | 上游 | 评估需要先整理案件事实 |
| 风险评估 | 核心 | 风险评估是评估报告的核心内容 |
| 尽职调查 | 上游 | 评估需要尽职调查的支持 |
| 合规审查 | 并行 | 合规审查是评估的一种特殊类型 |

## 六、局限性

第一，法律评估结论受到评估时点的法律法规有效性的限制，法律法规的变化可能影响评估结论。第二，评估的准确性依赖于提供的信息的真实性和完整性，信息缺失可能导致评估偏差。第三，法律问题存在固有不确定性，不同的裁判者可能对同一问题有不同的判断结论。第四，评估报告的时效性有限，情况变化可能影响评估结论的有效性。第五，风险评估中的概率判断具有主观性，不能等同于实际结果。
