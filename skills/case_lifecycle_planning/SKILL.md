## 概述

| 项目 | 内容 |
|------|------|
| **能力名称** | 案件生命周期管理 |
| **核心功能** | 对案件从收案到结案的全生命周期进行规划、跟踪和管理 |
| **适用领域** | 诉讼案件、仲裁案件、非诉项目、合规调查项目 |
| **管理阶段** | 收案评估、收案登记、案件策划、案件执行、结案归档 |
| **核心原则** | 系统化、及时性、完整性、可控性 |

## 法律声明

> **重要提示：** 本技能提供的案件生命周期管理辅助功能，不构成法律意见。使用者应注意：第一，案件管理计划应当根据案件的具体情况灵活调整，不能机械套用。第二，管理节点的时间安排应当以法律规定和法院通知为准，本技能提供的参考时间不替代法定期限的独立核实。第三，案件风险评估结论仅供参考，最终策略应当由主办律师决定。

## 一、核心概念

### 1.1 案件生命周期的阶段划分

案件生命周期可以分为以下阶段：收案评估阶段，对是否接受委托进行评估，包括利益冲突检索、案件可行性评估和风险评估。收案登记阶段，完成委托手续的办理，包括签订委托代理合同、收取律师费、整理案件基本信息。案件策划阶段，制定案件处理方案，包括诉讼策略、证据收集计划、时间安排和团队分工。案件执行阶段，按照策划方案推进案件，包括文书起草、证据整理、庭审准备和出庭代理。结案归档阶段，案件终结后完成结案报告、档案整理和费用结算。

### 1.2 各阶段的关键管控点

收案评估阶段的关键管控点包括：利益冲突检索是否完成、案件可行性评估报告是否出具、风险评估是否充分、收费方案是否确定。收案登记阶段的关键管控点包括：委托合同是否签署、当事人信息是否准确、案件编号是否分配、案卷是否建立。案件策划阶段的关键管控点包括：诉讼策略是否明确、证据清单是否完成、时间表是否制定、团队分工是否合理。案件执行阶段的关键管控点包括：各项文书是否按时提交、各项期限是否遵守、庭审准备是否充分、客户沟通是否及时。

## 二、操作步骤

### 2.1 收案评估

接受委托前应当进行全面的评估。首先进行利益冲突检索，确认是否存在利益冲突情形。然后对案件进行初步评估，分析案件的事实基础、法律依据和胜诉可能性。评估诉讼风险和成本，包括诉讼费、律师费、时间成本和可能的赔偿风险。确定收费方案，与客户就收费标准和方式达成一致。

### 2.2 收案登记

签订委托代理合同后，应当及时办理收案登记。登记内容包括：案件名称和编号、当事人信息（名称、地址、联系方式）、承办律师团队信息、收案日期和收费信息。建立案件档案，包括电子档案和纸质档案。

### 2.3 案件策划

制定案件处理方案。明确案件的核心目标和策略方向。梳理案件的法律关系和争议焦点。制定证据收集计划和时间表。安排团队分工和工作职责。

### 2.4 进度跟踪与调整

建立案件进度跟踪机制。定期更新案件进展状态。识别和处理案件执行中的问题和风险。根据案件进展调整策略和计划。

### 2.5 结案归档

案件终结后及时办理结案。完成结案报告，总结案件处理过程和结果。整理全部案件档案，按归档要求分类编号。完成费用结算，退还剩余预付款。收集客户反馈意见。

## 四、质量检查清单

收案评估检查：利益冲突检索已完成，案件可行性和风险已评估，收费方案已与客户确认。收案登记检查：委托合同已签署，当事人信息准确完整，案卷已建立。案件策划检查：诉讼策略明确，证据收集计划完成，时间表制定，团队分工合理。执行阶段检查：各项文书按时提交，期限遵守，客户及时沟通。结案检查：结案报告完成，档案整理归档，费用结算完成。

## 六、局限性

第一，案件生命周期管理计划可能因案件的特殊情况需要调整。第二，不同案件类型的生命周期管理要求不同，需要针对性调整。第三，管理工具的使用效果依赖于团队的执行力。
## 三、案件生命周期各阶段详述

### 3.1 收案评估阶段

收案评估是案件生命周期的起点，对案件处理的质量和效率具有决定性影响。评估内容包括：案件的基本事实是否清楚、法律关系是否明确、证据是否充分、诉讼时效是否届满、管辖法院是否适格、对方当事人的资信状况如何、案件的难易程度和预计工作量等。评估结论应当书面记录，作为是否接受委托的决策依据。

### 3.2 案件策划阶段

案件策划是案件管理的关键环节。策划内容应当包括：案件的核心争议焦点分析、诉讼请求或答辩策略的确定、证据收集和整理计划的制定、法律检索和研究计划的安排、关键时间节点的规划、团队分工和工作计划的制定、成本预算和风险控制的安排。

### 3.3 执行与监控阶段

执行阶段是案件生命周期中最长也最复杂的阶段。需要持续关注：案件进展是否按照计划推进、关键期限是否得到遵守、证据收集是否充分、对方当事人的动向和策略变化、法院的程序安排和裁判倾向、案件风险和不确定性的变化。

## 五、局限性

第一，案件生命周期管理的效果依赖于管理人员的专业素质和执行力。第二，不同类型的案件和客户对管理的要求差异较大，需要灵活调整。第三，管理工具和系统的使用不能替代专业判断和经验。第四，突发情况可能导致管理计划需要重大调整。
