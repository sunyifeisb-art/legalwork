# 法律尽职调查

## 概述

| 项目 | 内容 |
|------|------|
| **能力名称** | 法律尽职调查 |
| **核心功能** | 对目标企业或项目进行全面系统的法律尽职调查，识别法律风险和合规问题，出具尽职调查报告 |
| **适用领域** | 股权收购、资产收购、合资合作、融资上市、并购重组、项目投资 |
| **调查范围** | 主体资格、股权结构、资产权属、业务合规、重大合同、劳动关系、诉讼仲裁、知识产权 |
| **关键原则** | 全面性（覆盖所有风险领域）、独立性（客观中立）、保密性、程序规范性 |

## 法律声明

> **重要提示：** 本技能提供的尽职调查分析为辅助性专业意见，不构成正式法律意见。使用者应注意：第一，尽职调查的范围和深度应当根据交易的性质、规模和风险状况确定，本框架为通用性框架。第二，尽职调查的结论依赖于目标企业提供的信息和文件的完整性和真实性，对无法获取的信息应当进行适当的假设和风险提示。第三，尽职调查中发现的法律问题应当由执业律师结合具体交易结构进行综合评估。第四，尽职调查报告中的风险评级和判断具有一定的主观性，不同调查者可能有不同的判断结论。第五，尽职调查具有时效性，报告中的信息和判断仅在报告出具日有效。

## 一、核心概念体系

### 1.1 尽职调查的层次

根据调查的深度和范围，尽职调查可以分为三个层次：基础尽职调查主要核查目标企业的基本信息和公开信息，适用于初步筛选和估值阶段；标准尽职调查对目标企业的主要法律事项进行全面审查，适用于一般的股权收购和项目投资；深度尽职调查对特定风险领域进行专项深入调查，适用于高风险或高价值的交易。

### 1.2 尽职调查的核心领域

主体资格调查：核查目标企业的设立合法性、注册资本实缴情况、存续状态、股权结构及演变、实际控制人、公司治理结构等。应当审查营业执照、公司章程、验资报告、股东协议等基础文件。

资产调查：核查目标企业拥有的土地使用权、房屋所有权、知识产权、机器设备、车辆等主要资产的权利状态，确认是否存在抵押、质押、查封、冻结等权利限制。

重大合同调查：审查目标企业的重要合同，包括购销合同、融资合同、担保合同、特许经营合同等，关注合同中的控制权变更条款、限制性条款和重大违约条款。

劳动人事调查：审查劳动合同签订情况、社保缴纳情况、高管薪酬安排、竞业限制协议、员工股权激励计划等。

诉讼仲裁调查：核查目标企业及重要子公司的正在进行的和潜在的诉讼、仲裁、行政处罚案件。

知识产权调查：审查商标、专利、著作权、商业秘密等知识产权的权利归属、有效期限、许可使用情况和侵权风险。

### 1.3 尽职调查的基本流程

调查准备阶段，组建调查团队，制定调查计划，准备尽职调查清单（DDL）和保密协议。现场调查阶段，进行资料审阅、实地走访、管理层访谈、政府部门核查。分析整理阶段，对收集的资料和信息进行梳理、分析和交叉验证，形成调查发现。报告撰写阶段，根据分析和评估结果撰写尽职调查报告。成果交付阶段，提交尽职调查报告并进行汇报。

## 二、操作步骤

### 2.1 第一步：调查准备

制定详细的尽职调查清单，根据交易类型和目标企业的行业特点，确定调查的范围和重点。收集目标企业的基本信息和公开资料，包括工商信息、行业监管信息、公开财务数据等。准备保密协议，确保调查过程中的信息保密。确定调查时间表和人员分工。

### 2.2 第二步：资料收集与审阅

向目标企业发出尽职调查清单，跟进资料的收集进度。对收集到的资料逐份审阅，记录审阅发现，标注需要进一步核实的问题。对资料进行编号和归档，建立资料索引表。

### 2.3 第三步：实地调查与访谈

根据需要安排实地走访，参观目标企业的主要经营场所和生产设施。对目标企业的管理层、法务负责人、财务负责人等关键人员进行访谈。访谈内容应当围绕尽职调查清单中的关键问题展开，记录访谈纪要。

### 2.4 第四步：公共信息核查

通过公开渠道核查目标企业的相关信息，包括工商登记信息、不动产登记信息、知识产权公告信息、涉诉信息（中国裁判文书网、执行信息公开网）、行政处罚信息、企业信用信息公示系统等。

### 2.5 第五步：分析与报告撰写

对所有调查发现进行汇总分析，识别法律风险和合规问题。对各项风险进行定级评估，提出风险缓释和应对方案。撰写尽职调查报告，报告应当结构清晰、结论明确、风险提示充分。

## 三、输出格式

### 3.1 尽职调查报告结构

首部：报告名称、出具日期、调查对象、调查范围、调查期限、使用限制和免责声明。概要：交易背景、核心调查发现、主要风险提示和总体评估结论。主体部分：按调查领域逐一展开，每个领域包括调查情况、发现的问题、风险评级、整改建议。附录：文件清单、人员访谈清单、重要文件摘要、法律检索报告等。

### 3.2 常见风险发现示例

| 风险领域 | 风险描述 | 风险等级 | 整改建议 |
|----------|---------|---------|---------|
| 股权结构 | 目标企业存在代持股权，实际控制人持股比例不清晰 | 高 | 要求披露实际权益所有人，清理代持关系 |
| 知识产权 | 核心专利将于明年到期 | 高 | 评估到期后的竞争影响和替代方案 |
| 重大合同 | 与供应商的排他性供货协议将在控制权变更时自动终止 | 中 | 与供应商协商修改或重新签署协议 |
| 劳动关系 | 部分员工未签订书面劳动合同 | 高 | 立即补签劳动合同，评估双倍工资风险 |
| 诉讼仲裁 | 存在尚未了结的产品责任诉讼 | 中 | 评估赔偿责任风险，考虑在交易文件中约定补偿机制 |

## 四、质量检查清单

- [ ] 调查范围已明确定义并与交易性质匹配
- [ ] 尽职调查清单已发出并跟进
- [ ] 收到的文件已全部审阅并归档
- [ ] 公共信息核查已完成
- [ ] 关键法律事项已覆盖（主体、资产、合同、诉讼、劳动、知识产权）
- [ ] 风险评级合理
- [ ] 报告的免责声明和使用限制已充分说明

## 五、关联技能

| 关联技能 | 关系 | 说明 |
|----------|------|------|
| 合同风险审查 | 并行 | 审查重大合同中的风险条款 |
| 合规审查 | 并行 | 评估目标企业的合规状况 |
| 法律研究 | 上游 | 调查前进行相关法律检索 |
| 风险评估 | 下游 | 对调查发现进行风险评估 |
| 交易架构设计 | 下游 | 调查结论影响交易架构设计 |

## 六、局限性

第一，尽职调查的深度和广度受限于调查时间和调查资源的约束，不可能覆盖所有潜在风险。第二，调查过程中可能存在目标企业不配合或隐瞒信息的情况，影响调查的完整性和准确性。第三，部分法律风险的判断涉及主观评价（如诉讼胜诉概率），不同专业人员的判断可能存在差异。第四，尽职调查具有强烈的时效性，报告出具后目标企业的状况可能发生变化。第五，跨境交易中的尽职调查还面临不同法域的法律差异、语言障碍和文化差异等挑战。
