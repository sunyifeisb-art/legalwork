# 法律文书起草

## 概述

| 项目 | 内容 |
|------|------|
| **能力名称** | 法律文书起草 |
| **核心功能** | 根据案件事实和法律依据，起草各类法律文书 |
| **适用领域** | 民事诉讼、刑事诉讼、行政诉讼、仲裁、非诉业务、公司法律事务 |
| **可起草文书类型** | 起诉状、答辩状、上诉状、再审申请书、代理词、辩护词、法律意见书、律师函、合同/协议、申请书、异议书、执行申请等 |
| **关键原则** | 格式规范、引用准确、论证充分、语言严谨、结构清晰 |

## 法律声明

> **重要提示：** 本技能提供的法律文书起草辅助功能，生成内容仅供参考。使用者应注意：
> 1. 生成的法律文书需经执业律师审核并签署后方可正式使用
> 2. 不同法院、仲裁机构可能有特定的格式要求和提交规范
> 3. 文书中引用的法律条文应核实其现行有效性
> 4. 文书中引用的案例应核实其真实性和相关性
> 5. 涉及重大利益的文书，建议结合具体案情进行定制化修改

## 一、核心概念体系

### 1.1 法律文书的分类

按功能和应用场景，法律文书可分为以下主要类别：

| 文书类别 | 典型文书 | 核心功能 | 适用阶段 |
|----------|---------|---------|---------|
| **起诉/仲裁类** | 民事起诉状、仲裁申请书、行政起诉状 | 启动诉讼或仲裁程序，明确诉讼请求和事实依据 | 立案阶段 |
| **答辩/反诉类** | 民事答辩状、反诉状、仲裁答辩书 | 对起诉进行回应，提出抗辩或反请求 | 应诉阶段 |
| **上诉/再审类** | 民事上诉状、再审申请书、申诉书 | 不服一审裁判，启动二审或再审程序 | 上诉阶段 |
| **代理/辩护类** | 代理词、辩护词、质证意见 | 在庭审中陈述代理意见或辩护意见 | 庭审阶段 |
| **申请/异议类** | 财产保全申请、证据调查申请、执行异议书 | 申请法院采取特定措施或对程序提出异议 | 各阶段 |
| **非诉法律意见类** | 法律意见书、尽职调查报告、合规审查报告 | 就特定法律问题出具专业分析意见 | 非诉阶段 |
| **律师函/通知类** | 律师函、催告函、解除通知函 | 以律师名义发出正式法律函件 | 诉前/协商阶段 |
| **合同/协议类** | 买卖合同、服务协议、保密协议、和解协议 | 设定法律关系，明确各方权利义务 | 交易阶段 |
| **执行类** | 执行申请书、执行异议书 | 申请法院强制执行或对执行提出异议 | 执行阶段 |

### 1.2 法律文书的基本结构

尽管各类文书的格式各有不同，但通用结构可概括为：

```
┌─────────────────────────────────┐
│        标  题  区                │
│  （文书名称 + 案号/编号）         │
├─────────────────────────────────┤
│        当事人信息                 │
│  （各方当事人基本情况）            │
├─────────────────────────────────┤
│   事实陈述 / 案件背景             │
│  （客观、清晰的事实描述）          │
├─────────────────────────────────┤
│   法律分析 / 论证                 │
│  （法律依据 + 逻辑推理）           │
├─────────────────────────────────┤
│   请求 / 结论                    │
│  （明确的诉讼请求或结论性意见）      │
├─────────────────────────────────┤
│   尾部                          │
│  （此致、具状人/代理人、日期）       │
└─────────────────────────────────┘
```

### 1.3 法律文书的通用质量标准

| 质量标准 | 具体要求 | 常见问题 |
|----------|---------|---------|
| **准确性** | 事实准确、法条引用准确、当事人信息准确 | 错别字、事实表述模糊、法条编号错误 |
| **完整性** | 文书要素齐全、不遗漏必要内容 | 缺少当事人信息、遗漏诉讼请求、未列明证据 |
| **规范性** | 格式符合法院/仲裁机构要求 | 排版混乱、未按模板格式、缺少签名栏 |
| **逻辑性** | 事实-法律-结论逻辑链条清晰 | 跳跃式推理、论证不充分、因果关系错乱 |
| **严谨性** | 用语准确、表述无歧义 | 使用模糊词汇、逻辑漏洞、自相矛盾 |
| **目的性** | 紧扣文书的使用目的和读者需求 | 偏离主题、过分纠缠次要问题 |
| **说服力** | 论证有力、重点突出 | 平铺直叙、未突出有利事实、未回应对方观点 |

## 二、分类操作规范

### 2.1 起诉状类

#### 2.1.1 民事诉讼起诉状

**基本格式：**

```
民事起诉状

原告：（姓名/名称，住所地，统一社会信用代码/身份证号，法定代表人/负责人）

被告：（姓名/名称，住所地，统一社会信用代码/身份证号）

诉讼请求：
一、……（明确、具体、可执行）
二、……

事实与理由：
……（客观陈述案件事实，分层次论述法律依据）

此致
XXXX人民法院

具状人：（签名/盖章）
    年  月  日

附：1. 本状副本 X 份
    2. 证据材料 X 份
```

**写作要点：**

1. **诉讼请求**：必须明确、具体、可执行
   - 错误的写法：「请求法院判令被告还钱」
   - 正确的写法：「请求判令被告向原告返还借款本金人民币XXXX元」
   - 金钱给付类请求应明确金额、币种、计算方式
   - 行为请求应明确行为内容、履行期限
   - 多个请求应区分主请求和备用请求

2. **事实陈述**：按时间顺序客观陈述，避免主观评价
   - 宜：「2024年3月5日，原告与被告签订《借款合同》，约定……」
   - 忌：「被告恶意拖欠借款，严重违约」

3. **法律依据**：引用具体法律条文支持诉讼请求
   - 「依据《中华人民共和国民法典》第六百七十五条之规定……」
   - 「参照最高人民法院(2023)最高法民终XX号民事判决书的裁判观点……」

4. **管辖法院**：确认具有管辖权
   - 合同纠纷：被告住所地或合同履行地
   - 侵权纠纷：侵权行为地或被告住所地
   - 不动产纠纷：不动产所在地

#### 2.1.2 刑事自诉状

**基本结构：** 自诉人信息 → 被告人信息 → 案由和诉讼请求 → 事实与理由 → 证人/证据清单 → 尾部

**特殊要点：**
- 明确指控的罪名和对应的法律条款
- 提供被告人构成该罪名的具体事实和证据
- 注明该案件属于自诉案件范围（告诉才处理或被害人有证据证明的轻微刑事案件）

### 2.2 答辩状类

**基本结构：** 答辩人信息 → 案由 → 答辩意见 → 事实与理由 → 尾部

**写作要点：**

1. **针对性回应**：逐项回应起诉状中的事实和请求
   - 「关于原告主张的XXX事实，答辩人认为与事实不符，理由如下：……」
   - 「关于原告提出的第X项诉讼请求，答辩人认为缺乏法律依据，理由如下：……」

2. **抗辩策略选择**：
   - 事实抗辩：否认对方主张的事实（举证责任在对方）
   - 法律抗辩：承认事实但主张不构成法律上的责任
   - 程序抗辩：主张程序违法或管辖权异议
   - 时效抗辩：主张超过诉讼时效

3. **反诉**：如存在反诉事由，应在答辩状中同时提出反诉

### 2.3 代理词/辩护词

**基本结构：** 审判长/审判员 → 代理/辩护立场 → 案件概述 → 核心论点 → 结论与请求 → 尾部

**写作要点：**

1. **明确立场**：在开头即明确代理人/辩护人的基本立场
2. **核心论点组织**：围绕争议焦点组织论点，每个论点独立成段
3. **证据与论证结合**：每个论点均有事实依据和法律依据
4. **回应对方观点**：主动回应对方可能提出的反对意见
5. **价值判断**：在合法框架内进行价值判断和利益衡量

### 2.4 法律意见书

**基本结构：** 标题 → 致送对象 → 出具背景 → 事实假设 → 法律分析 → 结论性意见 → 尾部

**写作要点：**

1. **事实假设**：明确意见书所依据的事实假设，注明「基于贵司提供的以下事实」
2. **分析方法**：采用IRAC/问题-规则-分析-结论的分析框架
3. **风险提示**：明确指出法律风险，评估风险程度
4. **备选方案**：提供多个可行的方案并分析优劣
5. **免责声明**：限定意见书的适用范围和使用目的

### 2.5 律师函

**基本结构：** 标题 → 函号 → 致送对象 → 委托关系说明 → 事实陈述 → 法律分析 → 要求/期限 → 尾部

**写作要点：**

1. **正式但不咄咄逼人**：语气坚定有力但留有余地，为后续协商保留空间
2. **明确要求**：清晰地提出具体要求（停止侵权、履行义务、赔偿损失等）
3. **设定期限**：给出合理的回复期限
4. **声明法律后果**：如不履行将采取的法律行动，但要避免过激言辞
5. **保留证据**：律师函的寄送和送达应保留证据（快递单、签收回执等）

### 2.6 合同/协议类

**基本结构：** 合同名称 → 当事人信息 → 鉴于条款 → 定义条款 → 具体权利义务 → 违约责任 → 争议解决 → 其他条款 → 签章页

**写作要点：**

1. **关键条款设计**：
   - 陈述与保证条款：明确各方在签约时的基本事实保证
   - 交割条款：明确交割条件、时间和方式
   - 赔偿条款：明确违约赔偿的范围、限额和程序
   - 不可抗力条款：定义不可抗力范围及法律后果

2. **争议解决条款**：明确约定仲裁或诉讼，仲裁需明确仲裁机构和仲裁规则

3. **控制防范**：
   - 注意平衡双方权利义务
   - 避免过重的违约条款
   - 加入合理的免责和限制条款

## 三、文书起草通用流程

### 3.1 理解需求

1. **明确文书用途**：起诉、答辩、申请、通知还是意见
2. **确认目标受众**：法官、仲裁员、客户、对方
3. **理解事实背景**：全面了解案件事实
4. **确定关键诉求**：明确需要达成的核心目标

### 3.2 收集材料

1. 收集案件事实材料
2. 检索相关法律法规、司法解释
3. 查找相似案例（类案检索）
4. 收集程序性信息（法院、案号、承办法官等）

### 3.3 组织框架

1. 确定文书结构
2. 列出需要涵盖的要点
3. 确定论证逻辑顺序
4. 准备证据清单和引用依据

### 3.4 起草

1. 按照框架逐部分起草
2. 确保每一段都有明确的目的
3. 事实陈述和方法分析交替进行但保持清晰区分
4. 保持语言风格一致

### 3.5 审核修改

1. **事实核对**：核对所有事实表述是否准确
2. **法律核对**：核对所有引用的法律法规是否现行有效
3. **格式核对**：核对格式是否符合要求
4. **逻辑核对**：核理论证逻辑是否严密
5. **语言核对**：核对语言是否规范、有无错别字

## 四、各类型文书起草要点速查表

| 文书类型 | 核心目的 | 关键内容 | 语言风格 | 常见错误 |
|----------|---------|---------|---------|---------|
| 起诉状 | 启动程序、明确请求 | 诉讼请求、事实与理由 | 客观叙述、清晰明确 | 诉讼请求不具体、事实与理由不分 |
| 答辩状 | 回应对方主张、提出抗辩 | 逐项回应、抗辩理由 | 有针对性、逻辑清晰 | 未逐项回应、过分情绪化 |
| 上诉状 | 推翻或变更原审判决 | 原审错误认定、上诉请求 | 论证严谨、态度坚定 | 未针对原判错误、重复一审观点 |
| 代理词 | 说服法官支持己方 | 争议焦点分析、法律适用 | 逻辑严密、重点突出 | 偏离争议焦点、论证冗长 |
| 法律意见书 | 提供专业分析意见 | 法律分析、风险提示 | 客观专业、谨慎保守 | 结论模糊、风险提示不足 |
| 律师函 | 传达法律立场和诉求 | 事实陈述、要求事项 | 正式但留有余地 | 语气过激、要求模糊 |
| 合同 | 设定权利义务 | 权利义务、违约责任 | 严谨精确、无歧义 | 定义不清、条款不平衡 |

## 五、实务经验与方法（团队实践精华）

### 5.1 答辩状写作：全面陈述事实的利他思维

**核心原则**：答辩状既要说对自己有利的事实，也要说对自己不利的事实。

**为什么要全面陈述事实？**

1. **法官天然不信任律师**：法官知道律师只会说对自己当事人有利的事实。如果我们同时说有利事实和不利事实，更容易取得法官信任。

2. **核心不利事实无法回避**：对己方不利的核心事实是绕不过去的，只能正面应对。与其回避，不如主动按照我们的叙事方式去描述和解释。

3. **利他思维——帮法官写判决书**：法院判决书的核心就是"事实认定"和"本院认为"两部分。如果我们以帮法官写好判决书事实认定部分的态度去写答辩状，法官审案所需的事实自然会被罗列进来，法官也愿意看我们的材料，更容易采信我们的观点。

**答辩状标准结构：**

```
第一部分：事实陈述
├── 全面、客观地陈述案件事实经过
├── 有利事实：正面陈述，充分展开
├── 不利事实：主动描述，按我方视角解释
└── 用"据悉/据委托人介绍"标注未经文件确认的事实

第二部分：答辩观点
├── 每个观点独立成小标题
├── 每个标题下分几个小观点，逻辑自洽
├── 方便法官直观看清主要观点
└── 案情简单时可省略事实部分，直接写观点
```

### 5.2 证据筛选7项原则

在决定一份材料是否作为证据提交时，应当依次考虑以下7个问题：

| 序号 | 原则 | 具体问题 | 判断标准 |
|------|------|---------|---------|
| 1 | **证明目的** | 这份证据能否达到我方证明目的？ | 不能证明则不用 |
| 2 | **必要性** | 是否有其他更有利的证据替代？ | 非必需则考虑替换 |
| 3 | **不利性** | 这份证据是否对我方有不利之处？ | 利大于弊才提交 |
| 4 | **形式完整性** | 是否有盖章、签字等形式要求？ | 缺乏则补充或放弃 |
| 5 | **范围控制** | 内容太多时能否仅截取相关页面？ | 仅提供必要部分 |
| 6 | **版本确认** | Word版是否有对应的盖章版/定稿版？ | 以盖章版为准 |
| 7 | **内容不可删改** | 节选时是否对页面内容进行了删改？ | 只能节选，不能删改 |

### 5.3 质证意见四要义

1. **围绕三性**：始终围绕证据的真实性、合法性、关联性进行质证
2. **回应证明目的**：是否认可对方的证明目的？简要陈述理由
3. **同时指出有利之处**：这份证据如果有能证明我方观点的地方，也要同时指出——"这份证据同时证明……"
4. **区分"对证据的意见"和"观点陈述"**：质证环节是对证据本身发表意见，不宜在此环节过多展开观点，否则易被法官打断

### 5.4 证据清单编写规范

- **证据名称**：文件全称或文号/缩写，**不是**对证明内容的归纳
- **产生时间**：多份文件可只写一份的产生时间
- **证明内容**：一般与起诉状/答辩状的事实描述一一对应，可直接拷贝
- **页码标注**：统一标在纸面右下方，字体醒目，方便翻页
- **页码范围**：同一份文件有多份时，注明份数，如"财务凭证（3份）"
- **节选标注**：如文件非全文，标注"（节选）"

## 六、质量检查清单

```markdown
### 基本检查
□ 当事人信息完整准确
□ 引用的法律条文准确且现行有效
□ 诉讼请求/请求事项明确具体
□ 事实陈述客观，无主观评价
□ 文书格式符合规范要求

### 答辩状专项检查
□ 事实部分是否全面陈述了有利和不利的事实？
□ 是否采用了利他思维——"帮法官写判决书事实认定部分"？
□ 是否用"据悉"标注了未经文件确认的陈述？
□ 不利事实是否正面应对而非回避？

### 证据清单专项检查
□ 证据名称是否使用了文件全称而非证明内容归纳？
□ 是否按照证据筛选7原则逐一审查了每份证据？
□ 页码标注是否清晰统一？
□ 节选部分是否标注了"（节选）"？
```

## 九、局限性

1. **专业性限制**：生成的法律文书需经执业律师审核修改后方可正式使用
2. **地域差异**：不同地区法院可能有特定的格式要求，需根据实际情况调整
3. **时效性**：法律条文的更新可能导致部分引用的依据失效
4. **个性化不足**：AI生成的文书可能缺乏对特定案情和策略的深度把握
5. **策略考量不足**：诉讼策略的选择需要结合案件具体情况和专业经验
