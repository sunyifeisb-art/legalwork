# 律师时间管理

## 概述

| 项目 | 内容 |
|------|------|
| **能力名称** | 律师时间管理 |
| **核心功能** | 帮助律师在多案件并行场景下进行时间规划、优先级排序和效率提升 |
| **适用领域** | 诉讼案件管理、非诉项目管理、团队协作 |
| **关键原则** | 二八法则（先做重要不紧急）、倒排工期、团队借力、寻找捷径 |

## 一、核心方法

### 1.1 按案件委托阶段推进

第一阶段（确定委托）：了解案情、客户沟通、基础调研→通过团队分工集中完成
第二阶段（已委托案件）：根据案件时间节点推进

### 1.2 倒排工期法

从最晚时间节点倒推各项工作：确定开庭日期后，基于此倒推法律文书提交、庭前准备、证据整理等各环节的完成时间，并打出富余量。

### 1.3 二八法则

先做重要但不紧急的事→再做重要且紧急的事→其次做紧急但不重要的事。把更多时间和精力用在完成重要的事情上。

### 1.4 团队接力（"扔猴子"理论）

接到工作后先想哪些可以分配给团队小伙伴，第一时间安排出去并设好时间点，等完成后自己再接力。

### 1.5 日程表管理

每个律师都要有一个日程表，详细记录每天甚至未来几天的安排。确定工作时间后预留出来，每天更新。

## 二、操作步骤

### 2.1 每日梳理

每天早晨梳理手头所有案件，确定：哪些是眼前要做的、哪些是不着急做的；做好分工和安排；更新日程表。

### 2.2 任务分解

对每项工作：拆解为若干步骤；预估每个步骤所需时间；打出富余量（20%-30%）；设定中间检查节点。

### 2.3 借力与捷径

善用AI工具；学会"一人行走得快，众人行走得远"的借力思维；不停寻找工作捷径，最终目的是提高效率。

### 2.4 应对突发插事

当正在做一件事突然插进急事时：评估紧急程度；如确实紧急，记录当前进度后切换；完成后尽快回到原任务。

## 三、质量检查清单

- [ ] 每日早晨已梳理手头案件优先级
- [ ] 倒排工期计划已制定
- [ ] 每个任务已拆解步骤并预估时间
- [ ] 富余量已打出
- [ ] 日程表已更新
- [ ] 可分配的工作已安排给团队成员
- [ ] 已识别可使用的AI工具或捷径

## 四、关联技能

| 关联技能 | 关系 | 说明 |
|----------|------|------|
| 案件管理 | 并行 | 时间管理是案件管理的核心组成 |
| 团队协作 | 并行 | 借力和分工需要团队协作能力 |

## 五、局限性

第一，时间管理方法需要长期坚持才能见效。第二，不同律师的工作风格差异较大，需要根据个人特点调整。第三，突发事件可能打乱计划，需要保持灵活性。
