# legalwork 软件入口说明

这个项目文件夹里，真正的桌面软件项目是：

```text
apps/desktop-legalwork/
```

打开、开发、测试桌面端都以这个目录为准。它里面的 `package.json` 是桌面软件的工程入口，应用名称为 `legalwork`。

主要目录关系：

- `apps/desktop-legalwork/`：legalwork 桌面软件本体。
- `apps/desktop-legalwork/legalwork/`：桌面软件内置的本地 Agent 运行包。
- `vendor/data-compliance-review-codex/`：数据合规 / 脱敏功能依赖资源。
- `document/`、`redaction/`、`skills/`：根目录保留的 OCR、脱敏与法律技能能力包。
- `projects/compliance/`：独立的数据合规项目，不是桌面软件入口。

除非明确说“打包 DMG”，否则不要把 `dist/` 里的旧安装包当作最新软件。
