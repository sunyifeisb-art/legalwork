# Data Controllers Registry Regulation

Jurisdiction: Türkiye
Region: TR
Citation: Regulation on Data Controllers Registry
Authority: KVKK Authority
Source URL: https://www.kvkk.gov.tr/Icerik/6635/Data-Controllers-Registry
Topics: accountability, notice_transparency
Keywords: VERBIS, data controllers registry, Turkey KVKK

## Summary
Turkish registry rules requiring certain controllers to register and maintain processing inventory information.

## Key compliance obligations
- Determine VERBIS registration obligation.
- Maintain accurate processing inventory.
- Update registry information when processing changes.
