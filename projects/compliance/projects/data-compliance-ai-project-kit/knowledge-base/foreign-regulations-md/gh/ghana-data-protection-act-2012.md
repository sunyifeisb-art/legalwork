# Data Protection Act, 2012

Jurisdiction: Ghana
Region: GH

Keywords: Ghana Data Protection Act, Act 843, Data Protection Commission

## Summary
Ghana's data protection act with principles, registration, data subject rights, security safeguards, and commission oversight.

## Key compliance obligations
- Register with the Data Protection Commission where applicable.
- Comply with data protection principles and rights.
- Maintain security safeguards and processor controls.
