# Health Information Privacy Code 2020

Keywords: Health Information Privacy Code, HIPC, health information, New Zealand

## Summary
New Zealand health information code modifying privacy principles for health agencies and health information.

## Key compliance obligations
- Handle health information under health-specific privacy rules.
- Provide access and correction rights.
- Protect and disclose health information only under permitted conditions.
